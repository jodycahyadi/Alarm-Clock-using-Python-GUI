============Final Project Python Basic======================
Kelas Big Data using Python by KOMINFO

Description:
Program alarm clock sederhana dengan tampilan GUI yang didesain menggunakan OOP.

How to Run:
1. Activate virtual environment untuk dapat mengakses beberapa package yang digunakan seperti; tkinter & pygame
2. Pastikan sound "update.wav" sudah dalam 1 folder dengan file python "Alarm Clock"
3. Run file python
4. Setelah tampilan GUI muncul, silahkan set time untuk alarm
5. Setelah berhasil diset, tunggu waktu untuk mencapai set time dari alarm
6. Alarm akan berbunyi dengan suara dari sound "update.wav"
